# 使用指南

本页只讲一件事：怎么从 0 到 1 把一个视频处理完成。

## 标准操作（4 步）

1. 打开首页，粘贴 B 站链接。
2. 点击“开始处理”。
3. 等待系统自动完成：下载 → 转写 → 总结。
4. 在历史记录里查看并导出结果。

## 你会看到什么

- 视频标题和时长
- 完整逐字稿
- AI 总结结果
- 下载按钮（`.txt` / `.md`）

## 成功标准

出现以下任一结果即表示流程正常：

- 页面中展示了可阅读的逐字稿
- 页面中展示了结构化总结
- 历史记录中出现该任务并可再次打开

## 常见使用场景

- **快速过视频重点**：先看总结，再决定是否看原视频。
- **做学习笔记**：导出 `.md` 后直接整理到知识库。
- **沉淀团队资料**：把任务历史作为内部内容资产归档。

如果处理失败，请看：[常见问题排查](troubleshooting.md)
